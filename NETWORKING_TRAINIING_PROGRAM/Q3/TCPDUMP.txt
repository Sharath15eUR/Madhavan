TCPDUMP is the packet sniffing tool like wireshark but in the form of command and not in the GUI. It comes with flexible options to work with as like filters , packet limiters , interface selectors
, exporting and importing wireshark packet capture files etc. 

1. INSTALLING TCPDUMP : 

maddy@maddy-IdeaPad-Slim-3-15IAH8:~$ sudo apt install tcpdump
[sudo] password for maddy: 
Reading package lists... Done
Building dependency tree... Done
Reading state information... Done
tcpdump is already the newest version (4.99.1-3ubuntu0.2).
0 upgraded, 0 newly installed, 0 to remove and 205 not upgraded.
